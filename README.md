# Guardian Convergence Loop - Slide Deck

**Pure. Simple. Beautiful.**

A minimal, elegant slide deck presentation system built with Next.js, React, and TypeScript.

## Features

- **Pure simplicity**: Only slides and navigation arrows
- **Keyboard navigation**: Arrow keys and spacebar
- **Beautiful design**: Dark cinematic aesthetic with gold accents
- **Type-safe**: Full TypeScript support
- **Zero configuration**: Just works

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

- `app/page.tsx` - Unified entry point with deck selection
- `components/slides/` - Slide deck component and types
- `lib/slides/` - Slide data (unified, sales, methodology decks)

## Decks

- **Unified Convergence Deck** (30 slides) - Complete presentation
- **Sales Convergence Deck** (10 slides) - Focused sales presentation
- **Methodology Deck** (20 slides) - Core methodology

## Principles

- **YAGNI**: Only what's needed, nothing more
- **JØHN**: Truthful, simple, validated
- **Simplicity**: Maximum signal-to-noise ratio

## Deployment

Deploy to Vercel with zero configuration:

```bash
vercel
```
